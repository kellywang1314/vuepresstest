# withChildren

# 属性

属性名称 数据类型 是否必填 备注
children JSX.Element 是
num number 否
offset { top: number, right: number} 否, 默认: {top: 0, right: 0}

# 使用

    <Badge.withChildren
    	offset={
    		top:0,
    		right:0,
    	}
    	num={0}
    >
    	<View>
    		<Text>文字</Text>
    	</View>
    </Badge>

# standalone

属性名称 类型 是否必填 备注
num number 否

# 使用

    <Badge.standalone
    	num={0}
    />
